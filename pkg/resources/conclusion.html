What's Next?

The Snappy Web Agent is now running as a background service and will automatically start on system boot.

Service Management

To manage the service, use these commands in Terminal:

# Check service status
sudo launchctl list | grep com.snappy.webagent

# Stop the service
sudo launchctl stop com.snappy.webagent

# Start the service
sudo launchctl start com.snappy.webagent

# View logs
tail -f /var/log/snappy-web-agent/stdout.log

Connecting

The agent will be available on the first available port starting from 8436. Check the logs to see which port it's using.
